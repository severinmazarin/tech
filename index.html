/ Background /

Here you can find a growing collection of instructions on how to do different technical things. Mostly related to retrocomputing. The reasoning behind the creation of this website is that I did a lot of stuff once, and then forgot how to do it a year later. So now I intend to keep this as a record, mostly for myself. But it might be of help for someone! But that's just an added bonus.

 

/ Using this site /

Please use the menu in the header to browse the various themes. More themes/guides will be added in the future!

/ Design cues and goals /

  • The Classic Macintosh look with a sprinkle of DOS UI shadows. Only using basic HTML elements with minimum CSS and two "colors": 000000 & ffffff (or more commonly known as black & white).
  • If using a browser with basic or non-existant support for CSS, please set the display font to Courier (or similar monospace serif) at 14 px size.
  • Should be easily browseable on a Classic Mac or a 386 PC.
  • I've tested the website code in Brave Browser 1.79.119 and Netscape Navigator 3.
  • HTTP implementation (none of that modern HTTPS nonsense please).
